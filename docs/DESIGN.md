# 设计文档 - 多功能抽签系统

## 设计理念

### 关键词
- **活泼**: 色彩明快，充满活力
- **可爱**: 圆润造型，友好亲和
- **专业**: 功能完善，操作便捷
- **趣味**: 动画丰富，互动性强

### 设计原则
1. 界面简洁，功能明确
2. 操作流程符合直觉
3. 视觉反馈及时清晰
4. 适配各种屏幕尺寸

## 组件设计

### 按钮

**主要按钮**
- 背景: `#FF6B6B`
- 文字: 白色
- 圆角: 12px
- 悬停: 背景变浅 + 轻微放大
- 点击: 轻微缩小

**次要按钮**
- 背景: 白色
- 边框: 2px solid `#FF6B6B`
- 文字: `#FF6B6B`
- 圆角: 12px

### 卡片

- 背景: 白色
- 圆角: 16px
- 阴影: `0 10px 15px rgba(0,0,0,0.1)`
- 悬停: 阴影加深 + 轻微上浮

### 输入框

- 背景: 白色
- 边框: 2px solid `#E8E8E8`
- 圆角: 12px
- 聚焦: 边框变 `#FF6B6B`

## 动画设计

### 过渡动画

```css
/* 淡入淡出 */
.fade-enter { opacity: 0; }
.fade-enter-active { opacity: 1; transition: opacity 300ms; }
.fade-exit { opacity: 1; }
.fade-exit-active { opacity: 0; transition: opacity 300ms; }

/* 滑动 */
.slide-enter { transform: translateX(-100%); }
.slide-enter-active { transform: translateX(0); transition: transform 300ms; }
```

### 抽签动画

**竹签风格**
- 摇晃动画: ±5deg 旋转，持续 1s
- 弹出动画: 从底部滑入，带弹性

**卡片风格**
- 翻转动画: Y 轴旋转 180deg，持续 0.6s
- 缩放动画: 从 0.8 到 1.0

**转盘风格**
- 旋转动画: 多圈旋转 + 减速
- 指针动画: 轻微震动

## 响应式设计

### 断点

```css
/* 手机 */
@media (max-width: 640px) {
  /* 单列布局 */
}

/* 平板 */
@media (min-width: 641px) and (max-width: 1024px) {
  /* 双列布局 */
}

/* 桌面 */
@media (min-width: 1025px) {
  /* 多列布局 */
}
```

### 移动端优化

- 按钮高度至少 44px
- 文字大小不小于 14px
- 触摸间距至少 8px
- 避免横向滚动

## 可访问性

- 支持键盘导航
- 适当的 ARIA 标签
- 足够的颜色对比度
- 支持屏幕阅读器
