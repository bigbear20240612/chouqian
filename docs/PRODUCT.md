# 产品文档 - 多功能抽签系统

## 产品概述

一个功能完善、界面活泼可爱的抽签系统，专为课堂教学活动设计。老师可以使用该系统进行各种抽签活动，增加课堂趣味性。

## 目标用户

- 小学/中学教师
- 培训机构讲师
- 活动组织者

## 核心功能

### 1. 多种抽签模式

#### 数字抽签
- 设置数字范围（最小值-最大值）
- 可一次抽取一个或多个数字
- 结果不重复（可配置）

#### 姓名抽签
- 支持导入学生名单
- 可一次抽取一名或多名学生
- 记录已抽取学生（避免重复）

#### 诗词抽签
- 预置古诗词库
- 随机抽取诗词展示
- 包含诗词内容和作者

#### 自定义抽签
- 用户自定义抽签内容
- 支持文本、CSV、Excel 导入
- 适用于各种场景

### 2. 批量导入功能

- **文本导入**: 粘贴换行分隔的文本
- **CSV 导入**: 上传 CSV 文件
- **Excel 导入**: 上传 Excel 文件
- 自动验证数据格式

### 3. 多种展示风格

- **竹签风格**: 传统中式设计，竹签摇晃效果
- **卡片风格**: 现代卡片设计，3D 翻转动画
- **转盘风格**: 经典抽奖转盘，旋转减速动画

### 4. 历史记录

- 记录每次抽签结果
- 按时间/项目查询历史
- 支持导出历史记录

### 5. 项目管理

- 创建多个抽签项目
- 编辑项目配置
- 删除不需要的项目
- 快速切换项目

## 用户界面

### 主界面
- 项目列表卡片展示
- 快速创建项目按钮
- 项目搜索和筛选

### 抽签界面
- 大型抽签按钮
- 三种风格切换
- 实时结果显示
- 历史记录侧边栏

### 管理界面
- 项目 CRUD 操作
- 批量导入数据
- 历史记录查询

## 使用场景

1. **课堂提问**: 随机抽取学生回答问题
2. **分组活动**: 随机分配小组
3. **奖励抽奖**: 课堂小礼品抽奖
4. **诗词学习**: 随机学习古诗词
5. **座位安排**: 随机安排座位
6. **游戏环节**: 各种课堂小游戏

## 非功能需求

### 性能
- 抽签响应时间 < 100ms
- 支持 1000+ 条数据导入
- 页面加载时间 < 2s

### 兼容性
- 支持现代浏览器
- 响应式设计（平板/手机）
- 支持 iOS/Android 移动端

### 安全性
- 数据加密传输
- SQL 注入防护
- XSS 攻击防护

## 部署方案

- **平台**: Vercel
- **数据库**: Vercel Postgres (Neon)
- **域名**: 待定
- **SSL**: 自动配置
